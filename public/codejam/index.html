Contest Structure
Overview: Code Jam consists of one or more mathematical problem to be implemented in the programming language of your choice within given time(4 hours) frame at the venue.



Problem: You will be provided one or more mathematical problem with detail description, input and output parameters. You will be able to access the problem on a website at the venue.



Submission: You will be required to mail your solution to aswathanarayana.n@husky.neu.edu or upload to link provided. All solutions should adhere to the input and output format specified with problem description. Code will be judged on correctness and performance.





Announcement of Winners: Winner of the Contest will be mailed once the submission is evaluated.
Prizes: Exciting prizes in store for the winners.
